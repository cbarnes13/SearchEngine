Detailed Description

Parses XML Document.

Parses the WikiBooks XML document using rapidxml libraries, collecting the document title, date, contributor's username, and contents. The document contents are then split up into individual strings for each word. Words that are stop words are removed, and the leftover words have their stems removed. Then frequency of duplicate words is calculated, and the document title, date, contributor's username, and words are sent to the Index Handler.

Can also display the contents of a page in the xml file.
